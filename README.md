# VoiceAI Assistant - Frontend

A professional web interface for AI-powered voice and text conversations. Features real-time chat, voice recognition, and text-to-speech capabilities.

## ✨ Features

- **🎤 Voice Recognition** - Speak naturally using browser speech recognition
- **💬 Real-time Chat** - Beautiful chat interface with typing indicators
- **🔊 Text-to-Speech** - AI responses are spoken aloud
- **⏹️ Stop Control** - Stop speech anytime with the stop button
- **📱 Responsive Design** - Works perfectly on desktop and mobile
- **🎨 Modern UI** - Professional gradient design with smooth animations

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **Vanilla JavaScript** - No frameworks, pure performance
- **Web Speech API** - Browser-native speech recognition
- **Fetch API** - RESTful communication with backend
